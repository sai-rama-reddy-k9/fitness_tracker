A full-stack web application for tracking workouts and fitness progress.

## Features
- User registration and authentication
- Workout logging with exercise, duration, and calories
- Personal statistics dashboard
- Social feed showing recent community activities
- Persistent data storage with SQLite database

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Authentication**: bcrypt password hashing
